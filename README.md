# Webpack4 + ReactCodeMirror Issue

## Package Details
- webpack @4.1.1
- uglifyJS-webpack-plugin @1.2.3
- react-codemirror @1.0.0

## Instructions
`npm install`

`npm run build`

Open index.html

Look at JS error in console.

Repeating the same steps with `webpack.mode = 'development'` shows a working codeMirror.
